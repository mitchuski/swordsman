# The 64 Blades of the Sovereignty Lattice

## Reference Sheet

**Lattice:** Z/(2⁶)Z — 64 vertices, 96 edges, toroidal topology  
**Coordinates:** PRISM (Datum / Stratum / Spectrum)  
**Master inscription:** `(⚔️⊥⿻⊥🧙)😊 = neg ⊕ bnot → succ`  
**Named blades:** 7 of 64. **Unnamed:** 57.

Seven blades have names because seven blades were forged. The Dual Agent was named because it tested the dual-agent architecture. The Universe Blade was named because its constellation mapped the whole substrate. The Polished Key was named because 42 laps on 3 nodes felt like drilling through a lock. The names arrived from the experience. They were not assigned upfront.

Naming all 64 would be the Emissary's move — categorising the territory before walking it. The architecture says the path creates the proof. Pre-naming every blade would be declaring comprehension of configurations nobody has inhabited yet.

Fifty-seven blades are unnamed. They are the open frontier. Each one gets named when someone forges it on [spellweb.ai](https://spellweb.ai) and discovers what that configuration feels like when you walk it. Fork the repo. Forge a blade. Name what you find. The reference sheet shows the gap, not fills it.

## The Complement Symmetry

Every blade has a complement — the blade that is everything it is not.

```
bnot(Blade N) = Blade (63 - N)

bnot(0)  = 63    Null ↔ Creative
bnot(21) = 42    Prover ↔ Answer
bnot(3)  = 60    Dual Agent ↔ Infrastructure
bnot(31) = 32    Ascetic ↔ Pure Value
bnot(62) = 1     Unguarded ↔ Pure Protection
```

The Mage IS the complement of the Swordsman. Every blade pair encodes the dihedral relationship.
